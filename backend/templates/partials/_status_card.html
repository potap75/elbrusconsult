{% load icons %}{% comment %} Centered status card used by thanks / pending / confirmed / unsubscribed / ratelimited pages. Combines a tile-gradient icon, a `font-display` heading, and a short paragraph inside a `card-glass` shell, all sitting on top of the `topographic-bg` so the page never feels empty. Context variables (all optional except `status_heading`): - ``status_icon`` — Heroicon name for the top icon tile. Defaults to ``"sparkles"`` when omitted. - ``status_eyebrow`` — small aurora-tinted label above the heading. - ``status_heading`` — main heading (escaped plain text). - ``status_lead`` — supporting paragraph below the heading. - ``status_tone`` — visual tone of the icon tile. Currently: ``"success"`` (default, aurora gradient), ``"warning"`` (amber-tinted), ``"info"`` (slate-tinted), ``"farewell"`` (muted aurora). - ``status_primary_cta_label`` / ``status_primary_cta_url`` optional solid summit-colored primary CTA. - ``status_secondary_cta_label`` / ``status_secondary_cta_url`` optional outlined secondary CTA. {% endcomment %}
