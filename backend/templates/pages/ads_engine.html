The Elbrus Ads Engine (OAuth application name: elbruscloud) is an internal tool built and operated by {{ SITE_NAME }} to manage our own advertising for {{ SITE_DOMAIN }}. It is used exclusively by {{ SITE_NAME }} employees; it is not a public product and has no external users or customers.

What the application does

  • Reporting. Pulls performance reports (impressions, clicks, cost, conversions) for our own Google Ads campaigns so we can review results.
  • Campaign management. Lets an authorized employee pause or enable our campaigns, adjust daily budgets, add keywords, and create responsive search ads. New ads are always created in a paused state and reviewed by a person before going live.
  • Site analytics context. Reads our own Google Analytics 4 and Google Search Console data so paid performance can be compared with organic traffic.

Why the application requests Google user data

The application requests the Google Ads API scope (https://www.googleapis.com/auth/adwords) so it can read reports from, and make the campaign changes described above in, the Google Ads account owned by {{ SITE_NAME }}. It signs in only with our own company Google account. It does not access any other Google account, does not manage advertising for third parties, and does not collect data about visitors to this website.

How the data is used and protected

  • Data retrieved from the Google Ads API is used solely to report on and optimize {{ SITE_NAME }}’s own advertising. It is not sold, shared, or used for any other purpose.
  • Report results are displayed to the authorized employee during a working session and are not stored in a separate database.
  • The application runs on infrastructure we own, is reachable only over HTTPS, and requires authentication on every request. API credentials are stored in a managed secrets vault.
  • Use of data received from Google APIs adheres to the Google API Services User Data Policy, including the Limited Use requirements.

Who can use it

Access is limited to designated {{ SITE_NAME }} employees. There is no public signup, and the application cannot be authorized against Google accounts outside our organization’s control.

Privacy policy

Our privacy practices, including how we handle personal information collected on this website, are described in our Privacy and cookie notice.
